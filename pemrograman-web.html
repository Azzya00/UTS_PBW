Pemrograman Web

Halaman ini berisi materi dasar tentang HTML, CSS, JavaScript, PHP, Mengontrol Alur Program, Perulangan, dan Array.

1. HTML

HTML digunakan untuk membuat struktur halaman web.

<h1>Halo, Dunia!</h1>
<p>Ini adalah paragraf pertama dalam HTML.</p>
<a href="https://www.google.com">Klik di sini</a>

2. CSS

CSS digunakan untuk mengatur tampilan elemen HTML.

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

3. JavaScript

JavaScript digunakan untuk membuat website lebih interaktif.

function sapaPengguna() {
    alert("Halo, selamat datang di website saya!");
}
        
        
4. PHP

        
PHP adalah bahasa server-side scripting yang digunakan untuk website dinamis.

        
<?php
    echo "Halo, ini dari PHP!";
?>
        

        
        
5. Mengontrol Alur Program

        
Penggunaan if-else dalam PHP:

        
<?php
$nilai = 75;
if ($nilai >= 80) {
    echo "Lulus dengan nilai A";
} elseif ($nilai >= 70) {
    echo "Lulus dengan nilai B";
} else {
    echo "Tidak lulus";
}
?>
        

        
        
6. Perulangan

        
Contoh perulangan dalam PHP:

        
<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Perulangan ke-$i 
";
}
?>
        

        
        
7. Array

        
Array adalah struktur data yang digunakan untuk menyimpan banyak nilai dalam satu variabel.

        
Contoh array dalam JavaScript:

        
let buah = ["Apel", "Mangga", "Jeruk"];
console.log(buah[0]); // Apel
